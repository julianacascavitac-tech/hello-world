# VetTrace AI – Aplicación Android para Clínicas Veterinarias
## Descripción del proyecto
VetTrace AI es una aplicación Android diseñada para clínicas veterinarias. Permitirá registrar pacientes (mascotas), almacenar historial médico y exámenes anteriores, y generar reportes simples. Además, incluirá un prototipo de inteligencia artificial para resumir notas de consulta a partir de texto ingresado manualmente.
## Exposición del problema
Las clínicas veterinarias pequeñas suelen usar registros en papel o sistemas básicos. Esto dificulta la trazabilidad de los pacientes y el acceso rápido a información médica. VetTrace AI busca ofrecer una solución digital sencilla que centralice los datos y mejore la organización.
## Plataforma
- Android Studio con Kotlin  
- Firebase para almacenamiento en la nube  
- GitHub Classroom para control de versiones  
- Librerías de NLP para el prototipo de resumen de notas  
## Interfaz de usuario e interfaz de administrador
- Usuario (clínica): ver lista de pacientes, historial médico y reportes  
- Administrador (veterinario): registrar pacientes, añadir exámenes y notas de consulta  
## Funcionalidad
- Registro de pacientes con datos básicos  
- Historial médico con exámenes y tratamientos  
- Notas de consulta ingresadas manualmente  
- Resumen automático de notas con IA (prototipo)  
- Exportación de reportes simples en PDF  
## Diseño (wireframes o esquemas de página)
- Pantalla principal: lista de pacientes  
- Ficha médica: historial y notas  
- Panel de notas: ingreso de texto y resumen automático  
- Reportes: exportación de historial  
# M5 Progreso de la aplicación de Android

## Descripción del proyecto
VetTrace AI es una aplicación móvil para clínicas veterinarias que centraliza la información médica de las mascotas. Su objetivo es reducir la demora en el acceso a datos clínicos por parte de los veterinarios y ofrecer a los dueños de mascotas un acceso organizado y sencillo a la información relevante.

## Problema empresarial
Los médicos veterinarios enfrentan retrasos al acceder al historial clínico, lo que ocasiona repetición de exámenes y pérdida de tiempo. Los dueños de mascotas también tienen dificultades para consultar resultados y recordatorios de manera clara.

## Rol de la innovación
- Digitalización de procesos clínicos en la nube.  
- Acceso diferenciado para veterinarios y dueños de mascotas.  
- Inteligencia artificial para transcripción automática de consultas.  
- Interfaces adaptadas a cada perfil de usuario.
